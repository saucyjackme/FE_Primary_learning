# css引入方式

## 1.1 内嵌式
css写在style标签中，
style虽然可以写在页面任意位置，通常约定写在head中

## 2.2 外联式
css写在一个单独的.css文件中
需要通过link标签在网页中引入

## 3.3 行内式
css写在标签的style属性中

# 标签选择器

## 4.4 注意点
结构：标签名 {CSS属性名：属性值;}
注意点:
1.标签选择器是一类标签，非单独某一个
2.标签选择器无论嵌套关系有多深，都能找到对应的标签


## 5.5 css适用场景
|  引入方式   | 书写位置  |  作用范围   | 使用场景  |
|  ----  | ----  |  ----   | ----  |
| 内嵌式  | css写在style标签中 | 当前页面  | 小案例 |
| 外联式  | css写在单独的css文件中，通过link标签引入 | 多个页面  | 项目 |
| 行内式  | css写在style属性中 | 当前标签  | 配合js使用 |
